# Other Form Categories

## IB Update (主持人手冊更新)

Phase: `ib_update`. Filed when the Investigator's Brochure is updated.

### SF082 -- 更新主持人手冊審查送審資料表
Cover sheet for IB update submission. Auto-filled from config.

### SF083 -- 更新主持人手冊申請表
Application form describing IB changes. Requires manual summary of updates.

### SF084 -- 多中心信函審查送審資料表 (conditional)
Added when `multicenter: true`. Cover sheet for multi-center communications.

### SF085 -- 多中心信函申請表 (conditional)
Added when `multicenter: true`. Multi-center letter application.

## Drug Import (專案進口)

Phase: `import`. Filed when importing investigational drugs for a clinical trial.

### SF066 -- 專案進口審查送審資料表
Cover sheet for import submission.

### SF067 -- 專案進口申請表
Import application with drug details, quantity, storage requirements.

### SF068 -- 專案進口受試者同意書
Import-specific consent form supplement.

### SF093 -- 專案進口許可證明
Import permit documentation. Always included.

**Note**: Drug import forms are only relevant for interventional clinical trials with investigational drugs not yet approved in Taiwan.

## Suspension (計畫暫停/提前終止)

Phase: `suspension`. Filed when a study is paused or terminated early.

### SF047 -- 計畫暫停/提前終止審查送審資料表
Cover sheet for suspension/termination submission.

### SF048 -- 計畫暫停/提前終止報告書
Report explaining:
- Reason for suspension/termination
- Current enrollment status
- Plan for enrolled subjects (continued follow-up, referral)
- Data handling plan

## Appeal (申覆案)

Phase: `appeal`. Filed when appealing an IRB decision.

### SF077 -- 申覆案審查送審資料表
Cover sheet for appeal submission.

### SF054 -- 申覆案申請表
Appeal application with:
- Original IRB decision being appealed
- Grounds for appeal
- Supporting evidence or revised materials
- Requested outcome
